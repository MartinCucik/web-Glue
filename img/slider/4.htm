Husqvarna DM280 Core Drill 110v

Ref: DM280

Pricing: (exc. VAT)

RRP: £1646.00
You Pay Our Price: £1234.00
  • Description
  • Documentation

The Husqvarna DM280 110v Core Drill is a Robust 110V core drill motor for core drilling from 50mm up to 350mm (14'') in diameter used in conjunction with the DS 250 (Max 250mm diameter) or the DS450 Husqvarna drill stands.

This newly developed drill motor from Husqvarna is extremely easy to operate. The DM 280 features an ergonomic handle that makes it easy to carry around. The controls are designed and positioned to further enhance ergonomics and to facilitate drilling. The machine has three gear stages, so you can find the optimal drilling speed, with an LED indicator to allow the operator to drill at maximum pressure.
